Minimal News

Sistema Multiagente com CrewAI para Geração de Notícias

Arquitetura do Sistema

Pesquisador

Especializado em encontrar notícias relevantes de tecnologia. Utiliza ferramentas de busca avançada para identificar uma notícia principal e seus possíveis desdobramentos.

→

Analista

Analisa tendências e implicações da notícia principal, explorando diferentes ângulos e perspectivas para criar uma cobertura completa.

→

Redator

Transforma as análises em artigos coesos, adaptando o conteúdo para diferentes aspectos da notícia principal.

Fluxo de Trabalho

1

Pesquisa Inicial

O agente pesquisador utiliza a API do Google Search para encontrar uma notícia tecnológica relevante de 2025.

2

Análise Multifacetada

A notícia é analisada sob 5 perspectivas diferentes, gerando ângulos únicos de cobertura.

3

Produção de Conteúdo

Cada ângulo é transformado em um artigo completo, com contextualização e análise aprofundada.

4

Revisão e Publicação

O conteúdo é revisado, formatado e publicado automaticamente no site.

Tecnologias Utilizadas

CrewAI

Framework para orquestração de agentes autônomos, permitindo colaboração eficiente entre diferentes especialistas AI.

LangChain

Utilizado para processamento de linguagem natural e integração com modelos de linguagem avançados.

Django

Framework web para gerenciamento do backend e interface com o usuário.
